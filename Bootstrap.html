Mi Página Responsiva

Título del contenido

Este es un ejemplo de cómo Bootstrap maneja la grilla responsiva.

En una pantalla pequeña (como un móvil), la imagen y el texto se apilarán uno encima del otro (col-12). En una pantalla más grande (md o superior), se colocarán uno al lado del otro, ocupando la imagen 1/3 del ancho y el texto 2/3.

Otro bloque de contenido

Bootstrap permite reordenar columnas fácilmente. En este segundo ejemplo, el texto aparece a la izquierda en escritorio (gracias a `order-md-1`), aunque en el código HTML está escrito después o antes dependiendo de cómo queramos estructurarlo para móviles.
